## 本文件夹用于配置生活娱乐页面信息

编辑`entertainment.json`文件，在最小单元的数组中填写照片相对路径与照片名称即可.

建议归档为`./照片娱乐/年份`样式的目录结构


格式如下：

```json
{
    "content":[
        "./以前/聚餐.jpg",
        "./2013年/三叠井森林公园.jpg",
        "./2013年/三叠井森林公园_2.jpg",
        "./2013年/三叠井森林公园_3.jpg",
        "./2013年/三叠井森林公园_4.jpg",
        "./2013年/三叠井森林公园_20130518.jpg",
        "./以前/同门小聚.jpg",
        "./以前/同门小聚2.jpg",
        "./2014年/c.jpg"
    ]
}
```
> 本页面 *不支持* 以HTML方式编辑
